DBMS Capstone Project

SocialSphere

A fully normalised Social Media Database Application built with MySQL, Node.js & an iOS‑inspired interface.

1

Setup the Database

Click the Setup tab → hit Initialize Database. This runs schema.sql + sample_data.sql automatically, creating all 11 tables & seeding 183 records.

2

Explore Tables

Go to the Tables tab. Click any chip — users, posts, follows, likes, etc. — to instantly browse live relational data from your MySQL database.

3

Run Advanced Queries

Go to Analytics. Pick a query card (Joins, Self-Join, Subquery, Aggregation, Having) → click Run Query to execute real SQL and see live results rendered in the table.

4

Insert Live Data

Go to Insert. Register new users (with phone number — demonstrates the 3NF multivalued attribute decomposition) and publish posts directly into the live MySQL tables.
